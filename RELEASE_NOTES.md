# 📝 Saucer リリースノート

## バージョン 2.0.0 - 正式版リリース準備完了 🎉

**リリース日**: 2026年5月30日

### 🆕 新機能

#### 1. タグ機能
- 投稿に最大5つのタグを追加可能
- 15種類の定義済みタグから選択
- タグでフィルタリングして投稿を検索
- タグをクリックして同じタグの投稿を表示

#### 2. ユーザープロフィール強化
- 登録番号の自動採番（No. 1から順番）
- 登録日の表示
- ユーザーの投稿履歴を表示
- プロフィールページからの投稿詳細表示

#### 3. 読者（フォロー）機能
- 気になるユーザーを読者として登録
- 読者タブでフォロー中のユーザーを一覧表示
- 読者リストから直接プロフィールへ遷移
- 最新投稿のプレビュー表示

#### 4. Supabase統合（正式版）
- Google OAuth認証
- データの永続化
- リアルタイムデータ同期
- Row Level Security（RLS）によるセキュリティ

#### 5. PWA対応
- スマホのホーム画面に追加可能
- オフラインキャッシュ対応
- アプリアイコン設定済み

### 🔄 変更・改善

#### UIの最適化
- 3タブ構成（ホーム、読者、プロフィール）に簡素化
- チャレンジ・マッチング機能を削除
- より直感的なナビゲーション

#### データ管理
- 100件の仮想投稿データでデモモード充実
- モックデータの品質向上
- リアルなユーザーインタラクション

#### パフォーマンス
- ページ遷移の高速化
- リアクション処理の最適化
- データフェッチングの効率化

### 🛡️ セキュリティ

- Row Level Security（RLS）実装
- Rate Limiting（投稿: 1分に1回、リアクション: 1分に10回）
- OAuth認証による安全なログイン
- プライベート投稿の適切なアクセス制御

### 📦 技術スタック

- **フロントエンド**: React 18.3, TypeScript
- **UIライブラリ**: Material UI 7.3
- **状態管理**: React Hooks
- **ルーティング**: React Router 7.13
- **バックエンド**: Supabase（PostgreSQL, Auth, RLS）
- **デプロイ**: Vercel（推奨）
- **PWA**: Vite PWA Plugin

### 🐛 バグ修正

- 投稿作成後のフォーム初期化
- リアクションの重複防止
- プロフィールページの表示崩れ修正
- タグフィルターの挙動改善

### 📚 ドキュメント

新規作成されたドキュメント：
- `PRODUCTION_DEPLOYMENT_GUIDE.md` - 本番デプロイの完全ガイド

### ⚠️ 破壊的変更

#### 削除された機能
- 企業チャレンジ機能
- マッチング機能

これらの機能は、アプリのコンセプトを「日常の気づきを共有する」ことに集中するため削除されました。

### 🚀 デプロイ方法

詳細は `PRODUCTION_DEPLOYMENT_GUIDE.md` を参照してください。

#### クイックスタート

1. **Supabaseセットアップ**
   ```bash
   # 1. Supabaseプロジェクト作成
   # 2. SAUCER_DATABASE_SCHEMA.sql を実行
   # 3. Google OAuth設定
   ```

2. **環境変数設定**
   ```bash
   cp .env.example .env
   # .envファイルにSupabase認証情報を記入
   ```

3. **ビルド＆デプロイ**
   ```bash
   pnpm install
   pnpm build
   # Vercelにデプロイ
   ```
